A basic plugin to integrate tweetboard into your wordpress site.

== Description ==
This is a very basic plugin which integrates Tweetboard into your Wordpress
installation.
Once you're signed up (and approved) and have installed this plugin, the only
configuration
you need is to enter your twitter username in the configuration page of the
plugin, to 
be up and running.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `tweetboard4wp` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your twitter username in the admin preference page for Tweetboard.



== Frequently Asked Questions ==

= Why doesn't tweetboard appear on the site? =
If you haven't entered your Twitter username, the plugin will
not work. Please check the configuration in the admin section.

= Can I customize the apperance or position of Tweetboard? =
Please refer to the Tweetboard help for any configuration options.

== What is Tweetboard ==
From their website: Tweetboard is a fun and engaging micro-forum type
application for your website.  It pulls your Twitter stream in near 
real-time (max 1 min delay), reformatting tweets into threaded conversations 
with unlimited nesting. Conversations that spun off the original conversation 
are also threaded in-line, giving your site visitors full perspective of 
what's being discussed.

To use Tweetboard with your website you need to sign up and be approved.
